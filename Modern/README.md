# Modern Projects (Minecraft 1.21+)

This directory contains all plugins and modules developed for the latest versions of **Minecraft (1.21 and above)**.

### Technical Specifications:
* **Target Java Version:** 23
* **API:** Paper API 1.21.x
* **Features:** Utilizes modern Minecraft features such as Components (Adventure API), Data Components, and latest Mojang Mappings.

### Development Note:
Projects here take full advantage of the latest Java LTS features. Ensure your local development environment (JDK) is up to date.
