# Legacy Projects (Minecraft 1.8.8)

This directory contains all plugins and modules specifically developed for **Minecraft 1.8.8**.

### Technical Specifications:
* **Target Java Version:** Java 8 (JRE 1.8)
* **API:** Spigot / Paper API 1.8.8-R0.1-SNAPSHOT
* **Compatibility:** Designed for older server environments and PvP-focused gameplay.

### Development Note:
When contributing to projects in this folder, ensure that you do not use modern Java features (like Records or Sealed Classes) to maintain backward compatibility with Java 8 environments.
